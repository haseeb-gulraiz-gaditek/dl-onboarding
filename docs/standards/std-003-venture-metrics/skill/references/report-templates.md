# Report Templates Reference

Templates for STD-003 metric reports.

---

## Monthly Venture Report Template

```markdown
# Delivery Speed Report — {YYYY}-{MM}

**Generated:** {date}
**Period:** {start_date} to {end_date}
**Mode:** {Standalone | Enhanced (STD-002 + STD-001)}
**Collection:** {Full (GitHub API) | Partial (git-only)}

---

## Summary

| Indicator | Value | Trend | Previous |
|-----------|-------|-------|----------|
| PR Lead Time (Median) | {X}h | {up/down/flat} {delta} | {prev}h |
| PR Cycle Time (Median) | {X}h | {up/down/flat} {delta} | {prev}h |
| Coding Time (Median) | {X}h | {up/down/flat} {delta} | {prev}h |
| PR Throughput | {N} PRs | {up/down/flat} {delta} | {prev} PRs |
| Deployment Frequency | {N} deploys | {up/down/flat} {delta} | {prev} deploys |
| Deployment Failure Rate | {X}% | {up/down/flat} {delta} | {prev}% |

## Core Metrics Detail

### 1. PR Lead Time

- **Median:** {X} hours
- **P90:** {X} hours
- **3-month rolling:** {X} hours
- **Decomposition:** Coding Time ({X}h) + Cycle Time ({X}h)

### 2. PR Cycle Time

- **Median:** {X} hours (business hours, weekends excluded)
- **P90:** {X} hours

### 3. Coding Time

- **Median:** {X} hours
- **P90:** {X} hours

### 4. Review Iterations

- **Average:** {X} iterations/PR
- **Distribution:** 0 iterations: {N}% | 1: {N}% | 2: {N}% | 3+: {N}%
- **Target:** 80% within 2 iterations → {MET/NOT MET} ({actual}%)

### 5. PR Approval Rate (First Pass)

- **Rate:** {X}%

### 6. Deployment Frequency

- **Total:** {N} deployments
- **Average interval:** {X} hours

### 7. Time Between Deployments

- **Average:** {X} hours
- **Median:** {X} hours

### 8. PR Throughput

- **Total:** {N} PRs merged
- **Human PRs:** {N} | **Bot PRs:** {N} (excluded from other metrics)

### 9. PR Size Distribution

| Category | Count | % | Target |
|----------|-------|---|--------|
| Small (<100 lines) | {N} | {X}% | — |
| Medium (100-400) | {N} | {X}% | — |
| Large (400-1000) | {N} | {X}% | — |
| XL (>1000) | {N} | {X}% | <10% |

### 10. Deployment Failure Rate

- **Rate:** {X}%
- **Failed:** {N} | **Rollbacks:** {N} | **Hotfixes:** {N}

## Extended Metrics (when STD-002/STD-001 present)

### Spec Compliance
- **Rate:** {X}%
- **SDD cycles completed:** {N}

### Freshness Score
- **Current specs:** {X}%
- **Stale:** {N} | **Very stale:** {N} | **Missing dates:** {N}

### OKR Progress
- **Average key result score:** {X}
- **On Track:** {N} | **At Risk:** {N} | **Behind:** {N}

---

*Report generated by STD-003 Venture Metrics. Per-engineer data available in metrics/internal/ (lead access only).*
```

## Cross-Venture Benchmarking Summary Template

```markdown
# Cross-Venture Benchmarking — {YYYY}-{MM}

**Generated:** {date}
**Ventures included:** {list}

---

## Venture Comparison

| Metric | Venture A | Venture B | Venture C | Median |
|--------|-----------|-----------|-----------|--------|
| PR Lead Time (Median) | {X}h | {X}h | {X}h | {X}h |
| PR Cycle Time (Median) | {X}h | {X}h | {X}h | {X}h |
| PR Throughput (normalized) | {X}/eng | {X}/eng | {X}/eng | {X}/eng |
| Deployment Frequency (normalized) | {X}/eng | {X}/eng | {X}/eng | {X}/eng |
| Deployment Failure Rate | {X}% | {X}% | {X}% | {X}% |
| Review Iterations (avg) | {X} | {X} | {X} | {X} |
| First Pass Approval Rate | {X}% | {X}% | {X}% | {X}% |

### Normalization Notes

- PR Throughput and Deployment Frequency are normalized per active engineer
- Team size: Venture A ({N}), Venture B ({N}), Venture C ({N})
- Project phase: {early/growth/mature} — context for absolute value differences
- Per-engineer data is anonymized — only aggregated, normalized values shown

---

*Per-engineer breakdowns are internal-only and not included in this summary.*
```

## Period-over-Period Comparison Template

```markdown
# Period Comparison — {period_1} vs {period_2}

**Generated:** {date}

---

| Metric | {period_1} | {period_2} | Delta | Trend |
|--------|------------|------------|-------|-------|
| PR Lead Time (Median) | {X}h | {X}h | {+/-X}h | {better/worse/flat} |
| PR Cycle Time (Median) | {X}h | {X}h | {+/-X}h | {better/worse/flat} |
| Coding Time (Median) | {X}h | {X}h | {+/-X}h | {better/worse/flat} |
| Review Iterations (avg) | {X} | {X} | {+/-X} | {better/worse/flat} |
| First Pass Approval | {X}% | {X}% | {+/-X}% | {better/worse/flat} |
| Deployment Frequency | {N} | {N} | {+/-N} | {better/worse/flat} |
| Time Between Deploys | {X}h | {X}h | {+/-X}h | {better/worse/flat} |
| PR Throughput | {N} | {N} | {+/-N} | {better/worse/flat} |
| XL PR % | {X}% | {X}% | {+/-X}% | {better/worse/flat} |
| Deploy Failure Rate | {X}% | {X}% | {+/-X}% | {better/worse/flat} |

### Trend Direction

- **Better** = metric moved in the desired direction (lower times, higher throughput, lower failure rate)
- **Worse** = metric moved against desired direction
- **Flat** = <5% change
```
